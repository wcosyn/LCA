1. BUILDING
=============

This project now uses CMake to build and install. Perform the following steps.
	* $> cd build;
	* $> cmake ..
	* $> make (-j[numOfProcessors])
	* $> make install

[numOfProcessors] is the number of processors you want to use for building, this flag is optional.
